## Ответы на вопросы

**1. Что делает fetch?**  
`fetch` — это современный метод JavaScript для выполнения HTTP-запросов к серверу. Он позволяет получать данные с API или загружать файлы без перезагрузки страницы. Возвращает объект `Promise`, который содержит ответ сервера.

**2. Зачем нужны async/await?**  
`async/await` упрощают работу с асинхронным кодом.  
- `async` помечает функцию как асинхронную, которая всегда возвращает `Promise`.  
- `await` позволяет "подождать" выполнения `Promise` и получить результат как обычное значение, не используя цепочки `.then()`.

**3. Как работает localStorage?**  
`localStorage` — это встроенное хранилище в браузере, которое позволяет сохранять данные в виде пары ключ-значение.  
- Данные остаются даже после перезагрузки страницы.  
- Методы:  
  - `localStorage.setItem(key, value)` — сохранить значение  
  - `localStorage.getItem(key)` — получить значение  
  - `localStorage.removeItem(key)` — удалить значение
